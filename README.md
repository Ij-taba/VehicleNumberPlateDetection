## 📌 Overview

The **License Plate Recognition System** is a deep learning-based solution designed to automatically detect and extract vehicle license plates from images and video streams. It leverages **YOLO** for fast object detection and **OpenCV** for post-processing, making it suitable for real-world applications such as parking management, traffic monitoring, and security surveillance.

---

## ✨ Features

- 🔍 **Real-time License Plate Detection** — Fast inference using YOLO architecture
- 🖼️ **Image & Video Stream Support** — Works on static images and live camera feeds
- 🧹 **OpenCV Post-Processing** — Extracts and refines detected plate regions
- 📦 **Modular Pipeline** — Clean separation of detection, extraction, and output stages
- ☁️ **Deployable via Flask** — REST API ready for integration
- 📊 **75% Detection Accuracy** on custom test set (20,000 images)
